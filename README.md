# MyGHAT Bot

A ChatGPT-like chatbot web app built with Flask + Groq + SQLite.

## Features
- ChatGPT-style chat UI
- Chat history stored in SQLite
- Left sidebar chat records
- File upload support
- Streaming response
- Custom system prompt
- Temperature control

## Tech Stack
- Python Flask
- Groq API
- SQLite
- HTML / CSS / JavaScript
